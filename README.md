# yeymi-meraris
1. Uso de Cookies

Las cookies son archivos de texto que el navegador almacena para recordar información del usuario. Se utilizan para guardar preferencias, datos de inicio de sesión o configuraciones, haciendo que la navegación sea más rápida y personalizada.

2. Creación de Sesiones

Una sesión es un proceso que mantiene al usuario identificado mientras utiliza una aplicación web. Se crea al iniciar sesión y permite acceder a distintas páginas sin tener que autenticarse nuevamente.

3. Eliminación de Sesiones

Eliminar una sesión significa finalizar el acceso del usuario al sistema. Esto ocurre al seleccionar la opción Cerrar sesión o cuando la sesión caduca por seguridad, evitando que otras personas puedan acceder a la cuenta.
